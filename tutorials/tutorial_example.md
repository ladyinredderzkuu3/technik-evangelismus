# Tutorial Beispiel

Dieses Beispiel zeigt, wie man ein einfaches Python Skript erstellt.

## Schritt 1
Erstelle eine Datei mit dem Namen `example.py`.

## Schritt 2
Füge folgenden Code hinzu:
```python
print('Hallo Welt!')
```

## Schritt 3
Führe das Skript aus:
```
python example.py
```

## Abschluss
Du hast erfolgreich dein erstes Python Skript erstellt!
